Complicity (cumplicidade)

For this academic project, each student was given a random word, so that we could create an animation that would convey its meaning. So, I had the idea of using the dots "i" to represent two individuals that, with time, become more and more accomplices and end up forming a unique, lasting and special bond.

This project was made in the second year of the Design and Multimedia course at FCTUC, for the curricular unit "Typography in Digital Media". This academic work was supervised by professors Nuno Coelho and Paul Hardman.
