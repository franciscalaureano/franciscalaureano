Feels Good

ui ux

mockups

mobile app

“Feels Good” is an app whose end goal is to increase the well-being of unmotivated and disheartened people.

“How can Feels Good increase the well-being of people?”
Great question! After some investigation and talking to a psychiatrist, in order to understandd what can motivate these people, we concluded that performing small tasks promotes feelings of joy, motivation and reward, making people more satisfied, happy and willing to be more productive. As a result, we decided to focus our application on performing these small tasks.

“Feels Good” recomends activities that are ideal for each user, so they can perform them when they are more discourage. In addiction, it is also possible to use the application as an agenda, as each user can add an activity that is not one of those suggested by the application.

To better understand this project you can watch this video.

This project was carried out in the third year of the Design and Multimedia course at FCTUC, for the curricular unit "Interactive Media Design", that has as a main objective the practical application of the concepts, tools and processes of Interaction Design. This academic work was supervised by Professor Jorge Cardoso, and was created in collaboration with Angela Laspriella and Mariana Ferreira.
